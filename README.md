# Rambda scripts

Contains most scripts required to build `Rambda/Rambdax` related files.

## Requirements

Both repos `selfrefactor/rambda` and `selfrefactor/rambda-scripts` should be on the same file system level.

## Script

### Dynamic TSToolbelt

`yarn toolbelt`

### Check before release

`yarn consume`
